# Slide View

> **Status: Coming Soon** — Slide view is built but not yet exposed in the UI. It will be enabled in a future release.

The Slide View presents a markdown document as a presentation deck. Slides are navigated with arrow keys or on-screen buttons, with a progress bar showing your position.

## How It Will Work

Separate slides with `---` (three hyphens on their own line):

```markdown
# Slide Title

Content for the first slide.

---

# Second Slide

Content for the second slide.

- Bullet point
- Another point

---

# Final Slide

Wrap up content here.
```

## Formatting Tips

- Each slide should have a `#` heading for the title
- Keep slides focused — one idea per slide
- Use bullet points for easy reading
- Code blocks, tables, and images work within slides
- The progress bar shows how far through the deck you are
- Keyboard arrow keys provide quick navigation
